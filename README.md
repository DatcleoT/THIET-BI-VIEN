# Hệ thống Quản lý Thiết bị - SEP HUST

Dự án quản lý và vận hành thiết bị dành cho Trung tâm SEP - Đại học Bách Khoa Hà Nội.

## 🚀 Công nghệ sử dụng
* **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
* **Backend & Database:** Supabase (Auth, Database, Storage)
* **Giao diện:** Modern UI với hiệu ứng Glassmorphism.

## 🛠 Tính năng chính
* **Xác thực:** Đăng nhập/Đăng ký tài khoản dành cho Sinh viên và Giảng viên.
* **Quản lý:** Xem danh sách thiết bị theo danh mục chuyên ngành.
* **Phân quyền:** * **Sinh viên:** Tra cứu thông tin, xem quy trình vận hành chuẩn.
    * **Giảng viên:** Thêm, sửa, xóa thông tin thiết bị và cập nhật hình ảnh.
